    # 区间调度 -- 中等

    有许多 [start, end] 的闭区间, 请设计⼀个算法, 算出这些区间中, 最多有⼏个互不相交的区间。

    示例 1

    ⽐如 intvs = [[1,3], [2,4], [3,6]]
    这些区间最多有两个区间互不相交, 即 [1,3], [3,6], intervalSchedule函数此时应该返回 2

    这个问题在现实中其实有很多应⽤场景, ⽐如你今天有好⼏个活动可以参加, 每个活动区间⽤
    [start, end] 表示开始和结束时间, 请问你今天最多能参加⼏个活动？

    思路

    1. 从可选区间 intvs ⾥, 选择⼀个 end 最⼩的区间x
    2. 把所有与x相交的区间从 intvs 中剔除
    3. 重复 1,2, 直到 intvs 为空, 之前选出的各种区间 x， 就是我们要求的结果
    var eraseOverlapIntervals = function(intervals) {
  if (intervals.length === 0) return 0;
  const sortArray = intervals.sort((a, b) => a[1] - b[1]);
  let count = 1;
  let xEnd = sortArray[0][1];
  for (const item of intervals) {
    const start = item[0];
    if (start >= xEnd) {
      count++;
      xEnd = item[1];
    }
  }
  return count;
}

// 测试
const count = eraseOverlapIntervals([[1, 2], [2, 3], [2, 3], [3, 4], [7, 8]])
console.log(count); // 4

    # 合并区间 -- 中等

    以数组 intervals 表示若干个区间的集合，其中单个区间为 intervals[i] = [starti, endi] 。请你合并所有重叠的区间，并返回一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间。

    示例 1：

    输入：intervals = [[1,3],[2,6],[8,10],[15,18]]
    输出：[[1,6],[8,10],[15,18]]
    解释：区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6].

    示例 2：

    输入：intervals = [[1,4],[4,5]]
    输出：[[1,5]]
    解释：区间 [1,4] 和 [4,5] 可被视为重叠区间。

    var merge = function (intervals) {
  intervals.sort((a, b) => a[0] - b[0]);
  let prev = intervals[0]
  let result = []
  for(let i =0; i<intervals.length; i++){
    let cur = intervals[i]
    if(cur[0] > prev[1]){
      result.push(prev)
      prev = cur
    }else{
      prev[1] = Math.max(cur[1],prev[1])
    }
  }
  result.push(prev)
  return result
};

    # 汇总区间 -- 简单

    给定一个无重复元素的有序整数数组 nums 。

    返回 恰好覆盖数组中所有数字 的 最小有序 区间范围列表。也就是说，nums 的每个元素都恰好被某个区间范围所覆盖，并且不存在属于某个范围但不属于 nums 的数字 x 。

    列表中的每个区间范围 [a,b] 应该按如下格式输出：

    • "a->b" ，如果 a != b
    • "a" ，如果 a == b

    示例 1：

    输入：nums = [0,1,2,4,5,7] 输出：["0->2","4->5","7"] 解释：区间范围是： [0,2] --> "0->2" [4,5] --> "4->5" [7,7] --> "7"

    示例 2：

    输入：nums = [0,2,3,4,6,8,9] 输出：["0","2->4","6","8->9"] 解释：区间范围是： [0,0] --> "0" [2,4] --> "2->4" [6,6] --> "6" [8,9] --> "8->9"

    示例 3：

    输入：nums = [] 输出：[]

    解法一：顺序

    var summaryRanges = function(nums) {
  let r = [], i = start = 0
  while (i++ < nums.length) {
    if (nums[i] - nums[i - 1] !== 1) {
      r.push(nums[start] + (i - start === 1 ? '' : '->' + nums[i - 1]))
      start = i
    }
  }
  return r
};
